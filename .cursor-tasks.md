### Schema Changes

#### PaymentFrequency Enum

```prisma
enum PaymentFrequency {
  DAILY
  WEEKLY
  BI_WEEKLY
  MONTHLY
  QUARTERLY
  SEMI_ANNUAL
  ANNUAL
  CUSTOM
}
```

#### PropertyGroup Model

```prisma
model PropertyGroup {
  id          String     @id @default(cuid())
  name        String
  description String?
  userId      String
  createdAt   DateTime   @default(now())
  updatedAt   DateTime   @updatedAt
  properties  Property[]
  user        User       @relation(fields: [userId], references: [id], onDelete: Cascade)
}
```

#### Updated Property Model

```prisma
model Property {
  // Existing fields...
  paymentFrequency PaymentFrequency @default(MONTHLY)
  customPaymentDays Int[]           @default([])
  dueDateOffset    Int              @default(0) // Days after period start
  propertyGroupId  String?
  propertyGroup    PropertyGroup?   @relation(fields: [propertyGroupId], references: [id])
  // Other existing fields...
}
```

#### Updated Lease Model

```prisma
model Lease {
  // Existing fields...
  paymentFrequency PaymentFrequency @default(MONTHLY)
  billingCycleStart DateTime
  billingCycleEnd   DateTime?
  nextBillingDate   DateTime
  // Other existing fields...
}
```
